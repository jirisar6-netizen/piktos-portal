# 🏛️ PIKTOS PORTAL by Synthesis studio

> **Asistivní komunikační technologie nové generace pro děti s poruchou autistického spektra.**

## 🌟 O PROJEKTU
**PIKTOS** není jen aplikace. Je to komunikační most navržený v rámci **Synthesis studio** pro překonávání bariér v každodenním životě. Projekt vznikl z hluboké vnitřní motivace poskytnout špičkový nástroj pro:

* **Daniela** – syna mé partnerky Kateřiny, kterému hledáme cestu k vyjádření jeho světa.
* **Štěpánka** – mého syna, kterému chceme dopřát ty nejlepší technologie pro rozvoj a silné rodinné pouto.

## 🚀 KLÍČOVÉ FUNKCE (V0.1.20)
Aplikace využívá moderní webové technologie optimalizované pro mobilní zařízení (Xiaomi 13T Pro):

* 🧠 **Smart Bar:** Inteligentní predikce karet podle denní doby a historie.
* 🎭 **Social Scripts:** Vizuální scénáře pro zvládání stresových situací (např. "U doktora").
* 🆘 **Guardian SOS:** Okamžitá pomoc přes nouzovou kartu s kontaktem na Jiřího Šára.
* ☁️ **Cloud Sync:** Bezpečné ukládání personalizovaného nastavení do Google Tabulek.
* 🖨️ **Print Engine:** Generování fyzických piktogramů pro laminaci.

## 🛠️ TECHNICKÝ STACK
* **Frontend:** React + TypeScript + Vite
* **Design:** Professional Glassmorphism / Tailwind CSS
* **Standard:** ARASAAC (Augmentative and Alternative Communication)
* **Hosting:** Vercel (CI/CD z GitHubu)
